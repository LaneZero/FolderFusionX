FolderFusionX v1.1.0
English:
2025/03/01

Changes:
-Fixed the issue with the "Excluded Folders" functionality to correctly filter specified folders
-Added a Reload button to update the display with new settings
-Improved dark mode support for input fields and form elements
-Fixed the icon reference issue in the footer section
-Updated repository URLs and project naming across the application
-Enhanced error handling for folder access
-Improved UI responsiveness for different screen sizes
-Fixed the local storage key naming issue to ensure persistent settings

Known Issues:
-Large repositories (over 10,000 files) may cause performance issues
-Some system folders may be restricted due to browser security policies
-GitHub API limitations apply (60 requests per hour without authentication)

Future Features:
-File content search
-Folder comparison
-Collaborative sharing
-Custom display themes
-Offline mode support
